#### Ferramentas:

Instale várias ferramentas de uma vez:

![Screnshot 3](https://github.com/Olliv3r/toolmux/blob/main/media/multi-options.jpg)

### Instalação:
```
apt update
apt upgrade
apt install python git -y
git clone https://github.com/Olliv3r/Toolmux ~/Toolmux
```

### Execução:
```
cd ~/Toolmux
chmod +x toolmux.py
./toolmux.py
```
